# erdos-2026-heston-model

## Tuning the Heston Model (Erdös Institute Quant Finance Project Summer 2026)

Maya Betzler, Mykhailo Kuian, Daniel Lee, Kevin Specht, and Ariana Soria

### Objectives

Our main objective is to test how different objective functions affect the error, robustness, and computational complexity of the Heston model. The model is useful because it doesn't assume constant volatility, and the hope is to determine which functions and parameters make it the most effective.

### Datasets

Data was taken from TSLA options spanning 3 expiration dates in 2023.

### Stakeholders

- Economists, stockbrokers, or anyone else looking for the best way to model market trends
